# Experiments

A collection of experimental code, prototypes, and mini-projects.

## Structure

Organized by language, each mini-project lives in its own self-contained folder with a README explaining what it does and how to run it.

```
python/          Python experiments
javascript/      JavaScript / TypeScript experiments
rust/            Rust experiments
webflow/         Webflow custom code experiments
notebooks/       Jupyter notebooks for data exploration
```

## Projects

| Project | Language | Description |
|---------|----------|-------------|
| [nested-lists](webflow/nested-lists/) | Webflow | Nested list structures in Webflow |

## Adding a New Experiment

1. Pick the right language folder (or create a new one)
2. Create a subfolder with a descriptive name: `python/my-experiment/`
3. Add a short `README.md` (see `python/.template/` for an example)
4. Include dependency files (`requirements.txt`, `package.json`, etc.)
5. Update this table with a one-liner description
